# Axienta AI Chatbot

Axienta is a simple deep-learning based chatbot project built in Python.  
It supports both **terminal (CLI)** and **Tkinter GUI** interfaces using the same chat logic.

This project was created for learning, experimentation, and understanding how chatbots work end-to-end.

---

## Features

- Chat with the bot in the terminal
- Chat using a simple Tkinter UI
- Colored chat messages (user & bot)
- Word-wrapped messages
- Loading screen with timer
- Clean and beginner-friendly code structure

---

## Requirements

- Python 3.9+
- Required libraries:
  - torch
  - transformers
  - tkinter (comes with Python)
  - pillow (for icons)
